1. Layout utama aplikasi adalah portrait.Layout tidak responsive sehingga membuka aplikasi dengan mode landscape akan merusak layout keseluruhan.
2. Untuk memutar lagu dan melakukan pause dapat dilakukan dengan 2 tombol yang sudah disediakan.
3. Untuk keluar dari menu about dapat dilakukan dengan menekan tombol back pada HP (Tidak disediakan tombol back pada aplikasi).

English Version
1. The main layout isn't responsive. Changing the layout into a landscape layout will make an error for the application.
2. To play or pause the music you can use the button provided in the main layout with the related label.
3. For "About" menu, you can use the back button provided by your Android to quit the menu.
4. Lyric will be scroll automatically and can't be adjust manually by the user.
